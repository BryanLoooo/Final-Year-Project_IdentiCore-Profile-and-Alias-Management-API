Frequently Asked Questions


What is IdentiCore?

IdentiCore is a modern web platform that allows users to securely manage multiple online identities and aliases in one place. Whether you're a freelancer juggling personal and professional accounts, a privacy-conscious user, or someone who maintains different personas across platforms (e.g., GitHub, LinkedIn, Reddit), IdentiCore helps organize, tag, and control them efficiently.

It serves as a central dashboard for visibility control, tag-based grouping, and alias metadata. The platform supports features like reminders, profile editing, context tagging (e.g., “anonymous”, “work”, “fun”), and alias visibility toggles for managing exposure.


Is my data secure?

Yes, IdentiCore is built with a strong focus on data privacy and security. Sensitive information is encrypted in transit using HTTPS and would be stored securely using encryption-at-rest best practices in a production environment. For authentication, passwords are hashed using bcrypt, ensuring that even if database access is compromised, raw passwords are never exposed.

While the demo version may not include all production-grade security layers, the architecture is designed to integrate secure authentication tokens (like JWT), protect against CSRF/XSS vulnerabilities, and store no plain-text credentials. Future releases aim to include Two-Factor Authentication (2FA), role-based access controls, and enhanced audit trails.


Can I update or delete my account?

Absolutely. IdentiCore gives you full control over your user data. You can:

  • Edit your profile details such as username, age, email, and address from the Settings page.
  • Adjust your reminder preferences for managing aliases.
  • Permanently delete your account and all associated data using the “Danger Zone” option.

Please note that deletion is irreversible. Once you delete your account, all stored aliases, tags, and reminders will be permanently erased from the database.


What if I forget my login credentials?

In the current version of IdentiCore, basic email-password authentication is used. If you forget your credentials, there's no automated recovery system yet. You would need to contact the administrator or re-register.

However, a future update will introduce full password reset functionality via email with secure token validation. We also plan to support OAuth login options (e.g., Google Sign-In, GitHub) to make credential management easier.


What types of aliases can I create?

You can create aliases for any online identity you manage — personal accounts, work profiles, anonymous personas, or project-specific handles. Each alias can be tagged with platforms (like Reddit, GitHub, TikTok) and contexts (e.g., casual, professional) and marked as public or private depending on visibility.

These aliases are displayed in your dashboard, grouped and filterable by tags, making it easy to manage many identities without confusion.


Will IdentiCore remain free?

The current version of IdentiCore is free to use and intended for personal or educational purposes. In the future, a tiered system may be introduced:

  • Free Tier: Basic alias management, profile controls, and reminders.
  • Pro Tier: Advanced analytics, additional storage, data export, and integrations.

Any changes in pricing will be communicated clearly ahead of time. Your feedback will help shape future plans.
